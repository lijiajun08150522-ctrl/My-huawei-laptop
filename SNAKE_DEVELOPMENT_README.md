## 📊 技术实现

### HTML5 Canvas
每个阶段的画布都是独立的Canvas元素，实时绘制：
- **画布尺寸**：300x200
- **网格大小**：20px
- **绘制元素**：网格线、蛇、食物、UI

### JavaScript逻辑
- **状态管理**：每个阶段维护独立的游戏状态
- **绘制函数**：`drawSnake()`, `drawFood()`, `drawScore()`
- **阶段配置**：`stages`对象定义每个阶段的功能
- **动态更新**：切换阶段时自动更新所有画布

### 动画效果
- **淡入动画**：`fadeIn` CSS动画
- **过渡效果**：所有元素0.3s过渡
- **自动播放**：3秒间隔自动切换

## 🔧 技术细节

### 数据结构
```javascript
// 蛇的数据结构
const snake = [
    { x: 5, y: 5 },  // 蛇头
    { x: 4, y: 5 },  // 蛇身
    { x: 3, y: 5 }   // 蛇尾
];

// 食物数据结构
const food = { x: 10, y: 5 };

// 游戏状态
const gameState = {
    snake: [...],
    direction: { x: 1, y: 0 },
    food: {...},
    score: 0,
    level: 1,
    isRunning: false
};
```

### 绘制函数
```javascript
// 绘制蛇
function drawSnake(ctx, snake) {
    snake.forEach((segment, index) => {
        if (index === 0) {
            ctx.fillStyle = '#2E7D32'; // 蛇头
        } else {
            ctx.fillStyle = '#4CAF50'; // 蛇身
        }
        ctx.fillRect(
            segment.x * GRID_SIZE + 1,
            segment.y * GRID_SIZE + 1,
            GRID_SIZE - 2,
            GRID_SIZE - 2
        );
    });
}

// 绘制食物
function drawFood(ctx, food) {
    const centerX = food.x * GRID_SIZE + GRID_SIZE / 2;
    const centerY = food.y * GRID_SIZE + GRID_SIZE / 2;
    const radius = GRID_SIZE / 2 - 2;

    ctx.beginPath();
    ctx.arc(centerX, centerY, radius, 0, 2 * Math.PI);
    ctx.fillStyle = '#F44336';
    ctx.fill();
}
```
