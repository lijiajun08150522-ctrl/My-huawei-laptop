# 📱 快速开始 - 任务管理器手机访问

## 一键启动

```bash
python start_with_qrcode.py
```

---

## 手机访问地址

```
http://192.168.1.72:5000
```

---

## 三个简单步骤

### 1️⃣ 启动服务
```bash
python start_with_qrcode.py
```

### 2️⃣ 扫描二维码
- 浏览器会自动打开二维码页面
- 用手机扫描即可

### 3️⃣ 开始使用
- 在手机上添加、完成、删除任务
- 查看统计信息
- 数据自动同步

---

## 如果404错误？

运行诊断工具：
```bash
python diagnose.py
```

确保：
- ✅ 手机和电脑同一WiFi
- ✅ 防火墙允许Python
- ✅ IP地址正确 (192.168.1.72)
