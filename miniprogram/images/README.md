# 图标资源说明

此目录需要放置以下图标文件（建议尺寸：81x81px）：

## 底部导航图标

### 任务标签
- `task.png` - 未选中状态
- `task-active.png` - 选中状态

### 添加标签
- `add.png` - 未选中状态
- `add-active.png` - 选中状态

### 统计标签
- `stats.png` - 未选中状态
- `stats-active.png` - 选中状态

## 图标设计建议

1. **任务图标** - 列表或清单图标
2. **添加图标** - 加号或铅笔图标
3. **统计图标** - 图表或饼图图标

## 如何获取图标

### 方式1：使用在线图标库
- IconFont（阿里巴巴矢量图标库）
- Flaticon
- Freepik

### 方式2：使用设计工具
- Figma
- Sketch
- Adobe Illustrator

### 方式3：使用临时占位图
开发阶段可以使用纯色方块或简单形状作为临时图标。

## 注意事项

- 所有图标必须是PNG格式
- 推荐透明背景
- 建议尺寸：81x81px
- 未选中和选中状态需要区分明显
