# 任务管理器 - 微信小程序

基于微信小程序的任务管理工具，支持任务的增删改查和统计分析。

## 功能特性

### 核心功能
- ✅ 添加任务 - 支持设置优先级和分类
- ✅ 任务列表 - 查看所有任务，支持完成和删除
- ✅ 任务统计 - 完成率、分类统计、优先级分布
- ✅ 智能提醒 - 待办超过5个时显示警告
- ✅ 本地存储 - 数据保存在微信本地存储

### 任务属性
- **ID** - 唯一标识符
- **描述** - 任务内容
- **状态** - pending/done
- **优先级** - High/Medium/Low
- **分类** - Work/Study/Life/General
- **创建时间** - ISO 8601格式
- **完成时间** - 完成时记录

## 开发指南

### 前置要求
1. 注册微信小程序账号
2. 下载并安装微信开发者工具
3. 获取小程序 AppID

### 运行步骤

1. 打开微信开发者工具
2. 导入项目，选择 `miniprogram` 目录
3. 填入 AppID
4. 点击"编译"预览

### 配置 AppID

在 `project.config.json` 中修改：
```json
{
  "appid": "your_appid_here"
}
```

## 使用说明

### 添加任务
1. 点击底部"添加"标签
2. 输入任务描述（最多200字）
3. 选择优先级（高/中/低）
4. 选择分类（工作/学习/生活/通用）
5. 点击"添加任务"

### 管理任务
1. 在"任务"标签查看所有任务
2. 点击复选框或"完成"按钮标记完成
3. 点击"删除"按钮删除任务
4. 底部显示"清除已完成任务"按钮

### 查看统计
1. 点击底部"统计"标签
2. 查看今日简报
3. 查看分类统计和优先级分布
4. 待办超过5个时显示警告

## 技术栈

- 微信小程序原生框架
- 微信云开发本地存储（wx.setStorageSync）
- 响应式布局
- 组件化设计

## 注意事项

1. **AppID配置**：需要替换为真实的 AppID
2. **图标资源**：需要在 `images/` 目录添加实际图标
3. **数据持久化**：数据保存在微信本地存储，清除缓存会丢失
