# 🐍 贪吃蛇游戏 - 快速开始

## 🎮 立即开始

```
打开 snake_game.html
按空格键开始游戏
使用方向键控制蛇
```

---

## 📋 游戏规则

| 项目 | 规则 |
|------|------|
| **得分** | 每吃一个食物 +10分 |
| **升级** | 每5个食物升一级 |
| **奖励** | 每升一级 +50分 |
| **速度** | 200ms → 60ms (10个关卡) |
| **失败** | 撞墙或撞自己 |

---

## ⌨️ 控制方式

| 按键 | 功能 |
|------|------|
| ↑↓←→ / WASD | 移动方向 |
| 空格键 | 开始/暂停 |
| P键 | 暂停/继续 |
| R键 | 重新开始 |

---

## 🎯 游戏提示

1. **不要急**：初期速度慢，慢慢移动
2. **规划路线**：预判蛇的位置，避免撞自己
3. **利用角落**：在角落转弯更安全
4. **专注**：保持专注，避免误操作
5. **记录最高分**：挑战自己的最佳成绩！
