# 任务管理器 - 手机访问指南

## 📱 如何在手机上使用

### 方法1：启动服务（推荐）

1. **安装依赖**
   ```bash
   cd D:\SummerProject
   pip install -r requirements.txt
   ```

2. **启动服务**
   ```bash
   # 方式A：双击运行
   双击 start_server.bat
   
   # 方式B：命令行运行
   python app.py
   ```

3. **查看访问地址**
   
   启动后会显示类似信息：
   ```
   ======================================================
   任务管理器 Web服务
   ======================================================
   
   服务地址:
      本机访问: http://127.0.0.1:5000
      局域网: http://192.168.1.100:5000
   
   手机访问:
      确保手机和电脑在同一WiFi
      在手机浏览器打开: http://192.168.1.100:5000
   
   防火墙:
      如无法访问，请允许Python通过防火墙
   
   按 Ctrl+C 停止服务
   ======================================================
   ```

4. **手机访问**
   - 确保手机和电脑在同一WiFi网络
   - 打开手机浏览器
   - 输入显示的局域网地址（例如：`http://192.168.1.100:5000`）

5. **添加到主屏幕**（推荐）
   - 浏览器菜单 → "添加到主屏幕"
   - 像原生APP一样使用

## 🌐 网络设置

### 检查WiFi连接

**Windows**:
```
设置 → 网络和Internet → 查看当前WiFi名称
```

**手机**:
```
确保连接相同的WiFi网络
```

### 查看电脑IP

**Windows命令行**:
```bash
ipconfig
找到"无线局域网适配器"的IPv4地址
```

**Flask服务自动显示**:
- 启动时会自动显示本机局域网IP
- 直接使用显示的地址即可

## ⚠️ 常见问题

### Q: 手机无法访问？

**解决方法**:
1. ✅ 确认电脑和手机在同一WiFi
2. ✅ 检查防火墙设置
3. ✅ 确认服务正在运行
4. ✅ 检查IP地址是否正确
5. ✅ 尝试关闭防火墙测试

### Q: 防火墙提示？

**Windows防火墙**:
- 点击"允许访问"
- 勾选"专用网络"和"公用网络"
- 点击"允许访问"

### Q: 端口5000被占用？

**解决方法**:
编辑 `app.py`，修改端口号：
```python
port = 8000  # 改为其他端口
```

## 💡 使用技巧

### 技巧1：快速启动
- 创建桌面快捷方式指向 `start_server.bat`
- 双击即可快速启动

### 技巧2：开机自启
- 将快捷方式放入启动文件夹
- `Win + R` → `shell:startup`

### 技巧3：固定IP
- 在路由器设置固定电脑IP
- 避免每次重启IP变化

### 技巧4：使用热点
- 如果WiFi不稳定，用电脑开热点
- 手机连接热点访问

## 🔧 进阶配置

### 修改端口号

编辑 `app.py`:
```python
port = 8000  # 改为您想要的端口
```

### 修改监听地址

仅本机访问：
```python
app.run(host='127.0.0.1', port=port)
```

局域网访问：
```python
app.run(host='0.0.0.0', port=port)
```

## 📞 故障排除

### 无法启动服务

**检查**:
```bash
# 检查Flask是否安装
python -c "import flask; print('OK')"

# 检查端口是否被占用
netstat -ano | findstr :5000
```

### 手机连接很慢

**解决**:
- 使用5GHz WiFi频段
- 靠近路由器
- 减少网络干扰
