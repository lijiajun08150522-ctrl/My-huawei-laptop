# 任务管理器 Web应用

现代化的Web任务管理应用，支持任务增删改查和智能统计分析。

## 📱 功能特性

### 核心功能
- ✅ **任务管理** - 添加、完成、删除任务
- ✅ **优先级设置** - 高/中/低三个优先级
- ✅ **任务分类** - 工作/学习/生活/通用
- ✅ **智能统计** - 完成率、分类统计、优先级分布
- ✅ **积压提醒** - 待办超过5个时显示警告
- ✅ **本地存储** - 数据保存在浏览器本地存储
- ✅ **响应式设计** - 完美适配手机和桌面

### 界面特点
- 🎨 现代化渐变色设计
- 📱 移动端优先的交互体验
- 🔄 流畅的页面切换动画
- 📊 直观的统计可视化
- ⚡ 快速响应的操作体验

## 🚀 快速开始

### 方式1：直接打开（推荐）

1. 在电脑上打开 `web/index.html` 文件
2. 使用浏览器（Chrome、Edge、Safari等）访问
3. 立即开始使用！

### 方式2：本地服务器（推荐开发测试）

使用Python启动本地服务器：

```bash
# Python 3
cd D:\SummerProject\web
python -m http.server 8000

# Python 2
cd D:\SummerProject\web
python -m SimpleHTTPServer 8000
```

然后访问：`http://localhost:8000`

### 方式3：部署到服务器

将 `index.html` 上传到任何Web服务器即可访问。

## 📋 使用指南

### 添加任务
1. 点击底部导航的"添加"按钮
2. 输入任务描述（最多200字）
3. 选择优先级（🔴高 🟡中 🟢低）
4. 选择分类（💼工作 📚学习 🏠生活 📌通用）
5. 点击"添加任务"

### 管理任务
1. **任务列表页**查看所有任务
2. 点击✓复选框标记完成/取消完成
3. 点击"完成"按钮标记任务为已完成
4. 点击"删除"按钮删除任务
5. 底部"清除已完成任务"批量清理

### 查看统计
1. 点击底部导航的"统计"标签
2. 查看完成率大图
3. 查看分类统计（带进度条）
4. 查看优先级分布
5. 待办>5时显示警告提示

### 快速操作
- **浮动按钮** - 任务列表页右下角+号快速添加
- **底部导航** - 快速切换任务/添加/统计页面
- **顶部统计** - 随时查看任务概况

## 💾 数据存储

- **存储位置**：浏览器LocalStorage
- **存储容量**：通常5-10MB
- **数据持久性**：长期保存，清除浏览器数据会丢失
- **跨设备**：不支持，每个设备独立存储

**注意**：建议定期导出重要数据（未来版本支持）

## 🎨 技术栈

- **纯HTML5** - 无需框架依赖
- **CSS3** - 现代化样式和动画
- **Vanilla JavaScript** - 原生JS实现
- **LocalStorage API** - 本地数据存储
- **响应式设计** - 适配各种屏幕

## 🌐 在手机上使用

### 方法1：微信打开（最方便）
1. 将 `index.html` 发送到微信文件传输助手
2. 在微信中点击文件
3. 使用微信内置浏览器打开

### 方法2：二维码访问
1. 将文件部署到服务器（如GitHub Pages）
2. 生成二维码
3. 手机扫码访问

### 方法3：局域网访问
1. 电脑和手机连接同一WiFi
2. 电脑启动本地服务器（见上方）
3. 手机浏览器访问 `http://电脑IP:8000`

## 🔄 数据格式

每个任务对象结构：
```json
{
  "id": 1,
  "description": "任务描述",
  "status": "pending",
  "priority": "Medium",
  "category": "General",
  "createdAt": "2026-01-25T10:00:00.000Z",
  "completedAt": null
}
```

## 📝 注意事项

1. **数据安全** - 清除浏览器数据会丢失所有任务
2. **隐私保护** - 数据仅存储在本地，不会上传
3. **浏览器兼容** - 建议使用现代浏览器
4. **网络需求** - 无需网络，完全离线可用
5. **存储限制** - 超过LocalStorage容量会提示
